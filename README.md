# Cipher AI

واجهة محادثة ذكية (HTML/CSS/JS) تدعم عدة مزودي API مع تنسيق ردود مخصص وهوية **Cipher Ai**.

## التشغيل محلياً

افتح `chat.html` في المتصفح، أو شغّل خادماً محلياً:

```bash
npx --yes serve .
```

## GitHub Pages

1. في المستودع: **Settings → Pages**
2. **Source:** Deploy from a branch
3. **Branch:** `main` / `/ (root)`
4. الرابط: `https://x5coder.github.io/Cipher-AI/`

## الملفات

- `chat.html` — الصفحة الرئيسية
- `js/main.js` — الواجهة والمحادثات
- `js/providers/` — OpenAI, Anthropic, DeepSeek, Gemini, OpenRouter

## مفتاح API

يُخزَّن محلياً في المتصفح فقط (`localStorage`). لا ترفع مفاتيحك إلى GitHub.
