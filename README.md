# Cipher AI – Multi Provider Smart Chatbot

مشروع مفتوح المصدر من تطويري عبارة عن شات بوت متقدم يدمج أكثر من مزود ذكاء اصطناعي في واجهة واحدة، مما يتيح تجربة مرنة وقوية للمستخدم.

يدعم المشروع التفاعل مع عدة نماذج ذكاء اصطناعي مثل:
- DeepSeek  
- ChatGPT (OpenAI)  
- Google Gemini  
- OpenRouter  

---

## 🚀 المميزات

- محادثة موحدة تجمع أكثر من مزود AI في نفس الشات  
- تجربة ذكية لاختيار أفضل رد من أكثر من نموذج  
- دعم تنسيقات احترافية لعرض النصوص والكود بشكل منظم وجذاب  
- إمكانية اختبار أي API Key ومعرفة المزود الذي يعمل عليه تلقائياً  
- مناسب للمطورين لاختبار المفاتيح بدون الحاجة لمعرفة مصدرها مسبقاً  
- واجهة بسيطة وسريعة وقابلة للتطوير  

---

## 💡 فكرة المشروع

يعمل المشروع كطبقة وسيطة (AI Router) تقوم بتوجيه الطلبات إلى أكثر من مزود ذكاء اصطناعي، ثم عرض أو مقارنة الردود داخل نفس المحادثة، مما يمنح المستخدم أفضل تجربة ممكنة.

---

## ⚙️ طريقة الاستخدام

قم بإدخال API Key لأي مزود مدعوم، وسيحاول النظام تلقائياً تشغيله وتحديد الخدمة التي يعمل معها بنجاح.
